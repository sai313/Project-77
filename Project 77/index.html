What is Visual Studio Code?
Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js.
How to download and set up Visual Studio Code?
1. Download Visual Studio Code Installer from here.
2. Once it is downloaded open the installer and install Visual Studio Code. This will take a few minutes.
3. Then run and make your projects.

What is a console screen?
Console is a screen where you display your output, and it is used for testing purposes.
How to open console window?
Right click on the web page and click on inspect and then click on console to open console window.
How to print on console?
Use "console.log("text")" to print on console window.
